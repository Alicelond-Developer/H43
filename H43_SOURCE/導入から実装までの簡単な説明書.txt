**Python 3.6以上のインストール**


**依存関係のインストールについて**

ターミナル（例：CMD）でこのコマンドを実行します。: ```pip3 install -U discord.py```(もしpip3がなければpipだけで試してみてください。)

もしエラーが表示された場合は ```python -m pip install -U discord.py```

それでも同じエラー（「コマンドが認識されていません」）が出る場合は、pythonをシステムの環境変数に追加して再度試してみてください。

その後、スクリプトを実行するために

    1. ターミナルを開いてください。
    
    2. 'python3 h43.py`を実行します（`python3`が動作しない場合は`py`または`python`を試してください）。
    
    3. あとは楽しんでください！(操作方法がわからない場合は、親ディレクトリに戻って、README.mdの指示に従ってください)


詳しい説明やヘルプは、必要な情報がすべて含まれているREADME.mdをお読みください。
